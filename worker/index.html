LDPC BER Server - Worker

You have reached the LDPC BER Server at ${CONFIG_HOST}:${CONFIG_PORT)!

This is part of a larger project implementing a bit error rate tester (BERT) for the set of LDPC codes which can be decoded using the Xilinx SD-FEC Hardware Cores that are available on the RFSoC devices.

If you're looking for the api, see /api/.
